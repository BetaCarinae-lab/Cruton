## Syntax!  
Syntax is very simple*  
  
To start, create [nameofyourfile].crtn  
and write the following lines of code:  
```
.entry:  
    out "Hello, World!"  
end  
```

and 'compile' using node ./js/main.js ./[nameofyourfile].crtn -o [nameofyourfile].js
